# SogCountTo

> 数字增长或降低时增加动画效果

## 安装

``` bash
$ npm install sog-count-to -S
```

## 使用

在 `main.js` 文件中引入插件并注册

``` bash
# main.js
import SogCountTo from 'sog-count-to'
Vue.use(SogCountTo)
```

在项目中用使用 SogCountTo


```js
<template>
  <SogCountTo :startVal="0" :endVal="1000" />
</template>
```
